# kabukabar

Yahoo Financeから指定した銘柄の株価を表示するシンプルなmacOSメニューバーアプリケーションです。

![macOSメニューバーに'AAPL'の株価を表示するスクリーンショット。右クリックで開いたコンテキストメニューに、銘柄の設定やアプリケーションの終了オプションが日本語で表示されています。](https://user-images.githubusercontent.com/1561955/291617260-2212269a-6536-4148-8926-7876774c2057.png)

## 機能

- **最新の株価:** ユーザーが指定した銘柄の株価をメニューバーに直接表示します。
- **銘柄のカスタマイズ:** Yahoo Financeの任意のティッカーシンボル（例: `AAPL`, `5244.T`）に対応しています。
- **素早いウェブアクセス:** 株価を左クリックすると、Yahoo Financeの銘柄詳細ページが即座に開きます。
- **シンプルな設定:** 右クリックでコンテキストメニューを開き、銘柄の変更やアプリの終了が可能です。
- **自動更新:** 株価は1時間ごとに自動的に更新されます。

## 動作環境

- macOS 14.5 以降
- 有効なインターネット接続

## インストールと使い方

1. [Releases ページ](https://github.com/fukunotaisuke/kabukabar/releases)から最新の `kabukabar.app.zip` をダウンロードします。
2. ファイルを解凍し、`kabukabar.app` を `/Applications` フォルダに移動します。
3. アプリケーションを起動します。メニューバーに株価（デフォルト: `AAPL`）が表示されます。

**操作:**
- **左クリック:** Yahoo Financeの銘柄詳細ページを開きます。
- **右クリック:** 以下のオプションを含むメニューを開きます:
    - `銘柄を設定...` (Set Symbol): 新しいティッカーシンボルを入力するダイアログを開きます。
    - `終了` (Quit): アプリケーションを終了します。

## データソース

このアプリケーションは、Yahoo Financeの公開APIからデータを取得します。

```swift
// 株価を取得するためのAPIエンドポイント
let apiUrl = "https://query1.finance.yahoo.com/v8/finance/chart/" + symbol
```
