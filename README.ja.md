# 株価Bar

株価を右上のメニューバーに表示するMacアプリです。

## デモ
<img width="574" alt="kabukabar" src="https://github.com/user-attachments/assets/5726448f-add2-4d7b-9873-4abc6ecd0583">

## 機能
- 指定の株価をメニューバーに表示
- メニューをクリックすると株価のWebサイトを開く

## 必要環境
macOS 14.5以降

## 使い方
1. アプリをダウンロードして起動する
2. 好きな銘柄コードを設定する
3. メニューバーに株価が表示されるのを確認
