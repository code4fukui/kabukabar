# kabukabar

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A macOS application that displays a specific stock price in the top-right menu bar.

## Features
- Displays the price of a specific stock in the menu bar
- Allows the user to click on the price to open the stock's webpage

## Requirements
- macOS 14.5 or later

## Usage
1. Download the latest release of the `kabukabar` application.
2. Run the application. It will appear as an icon in the menu bar.
3. The application will automatically fetch and display the price of the configured stock.
4. Click on the stock price in the menu bar to open the stock's webpage.
